OpenAI 图像生成能力的重大升级

无论是需要快速生成概念草图，还是创建精细的商业素材，GPTImage2 都能满足您的需求。它的多模态理解能力使其能够准确把握用户意图，生成符合预期的图像作品。

常见问题

关于 GPTImage2 的常见问题

解答您最关心的问题

GPTImage2 是基于 GPT-4o 模型构建的图像生成系统，相比 DALL-E 系列在指令跟随、细节表现和生成速度方面都有显著提升。GPTImage2 与 ChatGPT 深度集成，可以通过自然对话进行图像生成和编辑，使用更加直观便捷。
GPTImage2 支持多种语言输入，包括中文、英文、日文、韩文、法文、德文等。建议使用英文描述以获得最佳效果，但中文描述也能生成高质量图像。
使用 GPTImage2 生成的图像版权归用户所有，可以用于个人和商业用途。但请注意不要使用 GPTImage2 生成受版权保护的角色、商标或其他受保护的内容。
ChatGPT Plus 订阅用户可以直接在 ChatGPT 界面中选择图像生成模式，然后输入文本描述即可生成图像。每月包含一定数量的生成额度，具体额度请查看您的订阅计划。
GPTImage2 API 按生成图像的数量计费。具体价格请参考 OpenAI 官方定价页面。API 适合需要大规模集成使用的企业用户和开发者。
GPTImage2 主要专注于静态图像生成。如果您需要视频或动画功能，可以关注 OpenAI 的其他产品更新。目前 GPTImage2 的强项在于高质量静态图像的生成和编辑。
提供详细、具体的描述是关键。包括主体、场景、风格、光线、氛围、视角等细节越丰富，生成的图像越符合预期。您也可以先生成初稿，然后使用局部编辑功能进行精细调整。
